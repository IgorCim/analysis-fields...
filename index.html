Что такое Анализполя.Ру?

Анализполя.ру - это универсальный помощник для исследования магнитного поля Земли. В его функционал входит анализ данных в формате Json: поиск максимальных и минимальных показателей, построение графиков, как по мировому, так и по локальному времени, построение графиков, как с одной, так и сразу с несколькими станциями, проверка файлов на работоспособность

Функционал: глобальное и мировое время

У Вас есть возможность строить графики как по мировому, так и по локальному времени. Если по мировому просто идёт анализ графиков, то локальное время позволяет узнать время суток для каждой станции, знание которого может помочь при анализе деятельности материи (работоспособность локального времени хорошо видна при построении графиков нескольких станций, каждый из которых будет сдвигаться на определённое число. Построение одного и многих графиков за раз

3. Построение одного и многих графиков за раз Существует 3 пути для построения графиков: "построить по папкам", "построить по папке" и "построить по файлу". Если Вы хотите, чтобы на одном графике была только одна станция, то "построить по файлу" будет кстати. Вам нужно файл с данными вставить в соответствующую папку. Если у Вас несколько файлов, которые никак между собой не корелируются, "построить по папке" будет необходима, она выдаст несколько графиков по одной станции в каждом. "Построить по папкам" будет полезна если вы захотите построить несколько станций на одном графике. Количество самих графиков зависит от количества самих папок, в которых будет по несколько файлов.

Максимальные, минимальные и диапазонные показатели

4. Максимальные минимальные и диапазонные показатели "Подсчёт данных" позволяет нам получить характеристики компонент и показателей. Эта функция позволяет нам получить самые большие и самые маленькие показатели, которые будут в файле, также функция будет считать разницу наибольшего и наименьшего. Ещё функция может "диапазонные" компоненты H, F, I и G, путём подстановки под показатели их разницу максимального и минимального числа. Расчёт идёт по формулам (North и East недоступны) (рис. ).
